首先确保服务器安装了`screen`，否则无法在后台运行服务端。

>其实nohup好像也可以，不过没试过。

文件1：start.sh

```bash
LD_LIBRARY_PATH=. ./bedrock_server
```

文件2：run_in_scr.sh

```bash
screen -dmS mcpe_1.14.60.5 ./start.sh
```

启动时，只需要运行`run_in_scr.sh`即可在screen中启动。
